# Homelab Project

## Networking
- **Switch**: MikroTik CRS328-24P-4S+RM switch acting as the router and managing VLANs.
- **VLANs**:
  - Personal management smartphones and laptops.
  - Kubernetes control planes and workers.
  - Proxmox VMs (Talos nodes, Windows VMs, etc.).
  - Guest WiFi devices.
  - IoT devices (coming from an Archer C7 1750 access point running OpenWRT via WiFi). No internet access
